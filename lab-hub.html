Adding a tool (for maintainers)
  1. Drop the tool’s HTML file into this same folder.
  2. Open index.html and add one entry to the TOOLS array near the bottom — name, desc, category, emoji, href, status.
  3. Set status to "live", "beta", "claude" (runs in Claude), or "soon". Re-host the folder. Done.
  4. Brand Assistant: publish the artifact in Claude, copy the share link, and paste it as that card’s href.
