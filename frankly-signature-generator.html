How to add it in Gmail

  1. Click Copy signature above.
  2. Click Open Gmail signature settings — it opens the right Gmail page in a new tab.
  3. Scroll to Signature → Create new → give it a name (e.g. “Frankly”).
  4. Click into the signature box and paste with Cmd/Ctrl + V.
  5. Set it as default for new emails and replies/forwards.
  6. Scroll down and click Save changes.
Logo not showing after paste?

Gmail normally uploads the logo for you. If it disappears: use Copy HTML code and ask Jonas to host the logo file at a fixed URL so we can swap the image for that link. Also make sure “Always display external images” is on in Gmail.

Why does the font look different from Figma?

Email clients can’t load Almarena Neue or Switzer — web fonts don’t work in email. So the signature uses Arial/Helvetica, which is what Gmail actually shows the recipient. Layout, colours, logo and hierarchy match the design.
